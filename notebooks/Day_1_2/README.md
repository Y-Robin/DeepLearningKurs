Die Tag-1-2-Inhalte sind in kleine, einzeln ausführbare Einheiten aufgeteilt. Die Reihenfolge folgt den Vortragsfolien und hält Datensatzsprünge möglichst klein.

| Nr. | Notebook | Folienbezug |
| --- | --- | --- |
| 01 | `01_tag_1_2_daten_samples_ausreisser_korrelation.ipynb` | Eingangsdaten, Samples, Ausreißer, Korrelation, Klassifikationsdaten |
| 02 | `02_tag_1_2_regression_mietpreise.ipynb` | Überwachtes Lernen, Regression, Regressionsmetriken |
| 03 | `03_tag_1_2_klassifikation_metriken.ipynb` | Klassifikation, Verteilungen, ROC, Confusion Matrix |
| 04 | `04_tag_1_2_train_valid_test_kreuzvalidierung.ipynb` | Training/Validierung/Test, Kreuzvalidierung, Hyperparameter |
| 05 | `05_tag_1_2_training_loss_regularisierung.ipynb` | Training, Loss, Bias-Variance, Regularisierung |
| 06 | `06_tag_1_2_unueberwachtes_lernen_clustering.ipynb` | Unüberwachtes Lernen, Clustering |
| 07 | `07_tag_1_2_reinforcement_learning_mini_pong.ipynb` | Reinforcement Learning, Agent, Environment, Reward |
