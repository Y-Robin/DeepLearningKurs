# Day 2/3 Notebooks

Die PDF ist als `Day 3` benannt, diese Beispiele sind aber für den zweiten Kursblock gedacht. Die Notebooks sind kleine, einzeln ausführbare Einheiten und folgen den Folien vom Neuron über Gradient Descent bis zu TensorFlow/Keras.

| Nr. | Notebook | Fokus |
| --- | --- | --- |
| 01 | `01_tag_2_3_neuron_scratch_klassifikation.ipynb` | Ein einzelnes Neuron von Hand bauen, manuell tunen und per Gradientenabstieg trainieren |
| 02 | `02_tag_2_3_gradient_descent_visualisierung.ipynb` | Loss-Kurve, Abstiegsweg und Effekt der Lernrate |
| 03 | `03_tag_2_3_lokale_minima_lernrate.ipynb` | Nichtkonvexe Loss-Landschaft, lokale Minima und Lernrate |
| 04 | `04_tag_2_3_keras_einstieg_training.ipynb` | TensorFlow/Keras importieren, Modell bauen, kompilieren, trainieren, evaluieren |
| 05 | `05_tag_2_3_mlp_dense_hyperparameter.ipynb` | MLP-Aufbau und wichtige `Dense`-Hyperparameter |
| 06 | `06_tag_2_3_aktivierungsfunktionen.ipynb` | ReLU, Sigmoid, tanh und Leaky ReLU vergleichen |
| 07 | `07_tag_2_3_batchnorm_dropout.ipynb` | Batch Normalization und Dropout gegen instabiles Training/Overfitting |
| 08 | `08_tag_2_3_skip_connections_functional_api.ipynb` | Skip Connections mit der Functional API |
| 09 | `09_tag_2_3_initialisierungen.ipynb` | Random, Glorot, He und Orthogonal vergleichen |
| 10 | `10_tag_2_3_loss_regularisierung.ipynb` | Loss in `compile`, Loss-Optionen und L1/L2-Regularisierung |
| 11 | `11_tag_2_3_optimierung_lr_scheduler_early_stopping.ipynb` | SGD, Momentum, RMSprop, Adam, Lernratenplan, Warmup und Early Stopping |
| 12 | `12_tag_2_3_batchsize_epochen.ipynb` | Batch Size, Epochen und Overfitting im Trainingsverlauf |
